# Source Layout

`src/` is the installable Python source root for `project-mai-tai`.

Top-level contents:

- `project_mai_tai/`
  - the real application package
- `project_mai_tai.egg-info/`
  - local packaging metadata generated by editable installs

Most code navigation should begin in:

- [project_mai_tai/README.md](./project_mai_tai/README.md)

Important separation:

- `src/project_mai_tai/` owns runtime logic
- `services/` owns only thin wrapper entrypoints
- `ops/` owns deployment and host operations
- `tests/` owns verification
- `sql/` owns Alembic migrations
