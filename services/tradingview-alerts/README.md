# TradingView Alerts Wrapper

Thin executable wrapper for `project_mai_tai.services.tradingview_alerts`.

Use when you want a familiar local launch command:

```powershell
python services/tradingview-alerts/main.py
```

The actual service logic lives in:

- `src/project_mai_tai/services/tradingview_alerts.py`
- `src/project_mai_tai/services/tradingview_alerts_app.py`
